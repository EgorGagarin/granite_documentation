Название:  granite (гранит) - макет приложения для бизнеса

Цель:
1. Обеспечение актуальной документацией
2. Сбор информации
3. Отслеживание показателей бизнес процесса

Общее описание:
1. Разработать приложение включающее (мобильное приложение, web версию, сервер)
2. Стек: java, Spring, PostgreSQL
3. Структура приложения в файле strukture
